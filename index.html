Estructura general

Lo primero que hay que hacer para crear una animación en css, es elegir el elemento div al que quieres aplicarle la animación y ponerle dos atributos clave: animation-name y animation-duration.

En animation-name te puedes inventar cualquier nombre que te guste para la animación y en animation-duration vas a indicar cuánto va a durar la animación. Por defecto, está propiedad está en 0, por tanto, si no se especifica una duración no se va a ejecutar.

Ahora, para definir la animación utilizamos la etiqueta @keyframes, seguida del nombre de la animación que hemos definido arriba y dentro de las llaves {} podemos definir el cambio que se va a realizar de dos maneras.

La primera forma de hacerlo es poner el atributo from seguido de la propiedad inicial que tiene el elemento, y el atributo to seguido de la misma propiedad cambiada.

div {

    /* aquí puedes poner los atributos que quieras */

    animation-name: MiPrimeraAnimacion;

    animation-duration: 7s;

}
@keyframes MiPrimeraAnimacion {

    from {background-color: red;}

    to {background-color: yellow;}

} 		Este circulo va a cambiar su color de rojo a amarillo.

Otra manera de hacerlo sería poner porcentajes por cada trámite que se desea cambiar.

@keyframes MiPrimeraAnimacion {

    0% {background-color: red;}

    50% {background-color: yellow;}

    100% {background-color: green;}

} 		Este circulo va a canbiar su color de rojo a amarillo y por último a verde.

Delay

Con esta propiedad se puede definir un retraso antes de empezar la animación. Puede ser tanto positiva, como negativa. Si es negativa, la animación va a empezar como si ya hubiera empezado hace X segundos.


@keyframes myAnimation {

    0% {background-color:red; left:0px;}

    50% {background-color:yellow; left:400px;}

    100% {background-color:green; left:0px;}

}




div {

    animation-name: myAnimation;

    animation-duration: 7s;

    animation-delay: 2s;

}



div {

    animation-name: myAnimation;

    animation-duration: 7s;

    animation-delay: -2s;

}

Direction

Esta propiedad permite cambiar el orden en el que se va a ejecutar la animación. Puede ser normal, al revers o alternada. Para indicarlo usamos las siguientes propiedades: normal (sin cambios), reverse (al revers), alternate (primero normal y después al revers) y alternate-reverse (primero al revers y después normal). Además, para ejecutar el alternate y alternate-reverse, hay que añadir la propiedad animation-iteration-count e indicar que se va a repetir 2 veces.


@keyframes myAnimation {
    0% {background-color:red; left:0px;}
    50% {background-color:yellow; left:200px;}
    100% {background-color:green; left:400px;}
}









div {

    animation-name: myAnimation2;

    animation-duration: 4s;

    animation-direction: normal;

}

div {

    animation-name: myAnimation3;

    animation-duration: 4s;

    animation-direction: reverse;

}

div {

    animation-name: myAnimation4;

    animation-duration: 4s;

    animation-iteration-count: 2;

    animation-direction: alternate;

}

div {

    animation-name: myAnimation5;

    animation-duration: 4s;

    animation-iteration-count: 2;

    animation-direction: alternate-reverse;

}
