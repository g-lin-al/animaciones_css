Estructura general

Lo primero que hay que hacer para crear una animación en css, es elegir el elemento div al que quieres aplicarle la animación y ponerle dos atributos clave: animation-name y animation-duration.

En animation-name te puedes inventar cualquier nombre que te guste para la animación y en animation-duration vas a indicar cuánto va a durar la animación. Por defecto, está propiedad está en 0, por tanto, si no se especifica una duración no se va a ejecutar.

Ahora, para definir la animación utilizamos la etiqueta @keyframes, seguida del nombre de la animación que hemos definido arriba y dentro de las llaves {} podemos definir el cambio que se va a realizar de dos maneras.

La primera forma de hacerlo es poner el atributo from seguido de la propiedad inicial que tiene el elemento, y el atributo to seguido de la misma propiedad cambiada.

div {

    /* aquí puedes poner los atributos que quieras */

    animation-name: MiPrimeraAnimacion;

    animation-duration: 7s;

}
@keyframes MiPrimeraAnimacion {

    from {background-color: red;}

    to {background-color: yellow;}

} 		Este circulo va a cambiar su color de rojo a amarillo.

Otra manera de hacerlo sería poner porcentajes por cada trámite que se desea cambiar.

@keyframes MiPrimeraAnimacion {

    0% {background-color: red;}

    50% {background-color: yellow;}

    100% {background-color: green;}

} 		Este circulo va a canbiar su color de rojo a amarillo y por óltimo a verde.

Delay

Con esta propiedad se puede definir un retraso antes de empezar la animación. Puede ser tanto positiva, como negativa. Si es negativa, la animación va a empezar como si ya hubiera empezado hace X segundos.


@keyframes myAnimation {

    0% {background-color:red; left:0px;}

    50% {background-color:yellow; left:400px;}

    100% {background-color:green; left:0px;}

}




div {

    animation-name: myAnimation;

    animation-duration: 7s;

    animation-delay: 2s;

}



div {

    animation-name: myAnimation;

    animation-duration: 7s;

    animation-delay: -2s;

}

Direction

Esta propiedad permite cambiar el orden en el que se va a ejecutar la animación. Puede ser normal, al revers o alternada. Para indicarlo usamos las siguientes propiedades: normal (sin cambios), reverse (al revers), alternate (primero normal y después al revers) y alternate-reverse (primero al revers y después normal). Además, para ejecutar el alternate y alternate-reverse, hay que añadir la propiedad animation-iteration-count e indicar que se va a repetir 2 veces.


@keyframes myAnimation {
    0% {background-color:red; left:0px;}
    50% {background-color:yellow; left:200px;}
    100% {background-color:green; left:400px;}
}









div {

    animation-name: myAnimation2;

    animation-duration: 4s;

    animation-direction: normal;

}

div {

    animation-name: myAnimation3;

    animation-duration: 4s;

    animation-direction: reverse;

}

div {

    animation-name: myAnimation4;

    animation-duration: 4s;

    animation-iteration-count: 2;

    animation-direction: alternate;

}

div {

    animation-name: myAnimation5;

    animation-duration: 4s;

    animation-iteration-count: 2;

    animation-direction: alternate-reverse;

}

Fill - mode

Esta propiedad especifica un estilo para el elemento seleccionado cuando la animación no se está ejecutando: antes de comenzar, después de terminar o ambas opciones al mismo tiempo. Puesto que las animaciones en CSS no afectan al elemento antes del primer fotograma animado, o después del óltimo, esta propiedad permite sobreescribir este comportamiento.

Los valores de esta propiedad son : none (la animación no aplicará ningón estilo al elemento antes o después de su ejecución), forwards (el elemento retendrá los valores de estilo fijados por el óltimo fotograma), backwards (el elemento retendrá en este caso los valores del primer fotograma), both (la animación seguirá las reglas de forwards y de backwards, extendiendo las propiedades fijadas en ambas direcciones), initial (reestablece la propiedad a sus valores por defecto) y inherit (hereda las propiedades del elemento padre del HTML).


div {

    animation-name: myAnimation;
    animation-duration: 5s;

}


#div1 {animation-fill-mode: none; background-color: red;}
#div2 {animation-fill-mode: forwards; background-color: orange}
#div3 {animation-fill-mode: backwards; background-color: yellow;}
#div4 {animation-fill-mode: both; background-color: rgb(64, 219, 25);}
#div5 {animation-fill-mode: initial; background-color: blue;}
#div6 {animation-fill-mode: inherit; background-color: rgb(250, 126, 147);}



@keyframes myAnimation {

    from {left: 50px;}
    to {left: 530px;}

}












Iteration - count

Esta propiedad especifica el nómero de veces que se va a repetir la animación. Puede tomar como valor un nómero específico de veces o infinite (infinito nómero de veces).


div {

    animation-name: myAnimation;
    animation-duration: 4s;
    animation-iteration-count: infinite;

}


@keyframes myAnimation {

    0% {background-color:red; left:100px; top:100px;}
    25% {background-color:yellow; left:480px; top:100px;}
    50% {background-color:blue; left:480px; top:350px;}
    75% {background-color:green; left:100px; top:350px;}
    100% {background-color:red; left:100px; top:100px;}

}

Timing - function

Otra función es animation-timing-function que especifica la velocidad de la animación en varios tramites, es decir, en una especie de curva de velocidad. Para definir los tramites tenemos varias opciones: ease (comienzo y final más lento que el recorrido), linear (la misma velocidad en todo el recorrido), ease-in (comienzo más lento), ease-out (final más lento), ease-in-out (comienzo y final más lentos) y cubic-bezier(n,n,n,n) (estableces los valores que quieres).


div {

    animation-name: myAnimation;
    animation-duration: 5s;

}


#div1 {animation-timing-function: linear;}
#div2 {animation-timing-function: ease;}
#div3 {animation-timing-function: ease-in;}
#div4 {animation-timing-function: ease-out;}
#div5 {animation-timing-function: ease-in-out;}
#div6 {animation-timing-function: cubic-bezier(0.075, 0.82, 0.165, 1);}



@keyframes myAnimation {

    from {left: 50px;}
    to {left: 530px;}

}
